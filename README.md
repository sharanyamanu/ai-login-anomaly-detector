
# 🚨 LogShield AI – Threat Intelligence Dashboard

🔗 Live Demo: https://ai-login-anomaly-detector-u1pp-sharanyamanus-projects.vercel.app
🔗 Backend API: https://ai-threat-backend.onrender.com/

## 📌 Overview
AI-powered system to detect anomalous login behavior using Machine Learning.

## ⚙️ Tech Stack
- Frontend: React, TypeScript, Tailwind
- Backend: Flask, Scikit-learn
- Deployment: Vercel + Render

## 🚀 Features
- Upload login logs (CSV)
- Detect suspicious activity
- Risk scoring system
- Interactive dashboard (charts + table)
- Downloadable report

## 🧠 How it works
Frontend → sends file → Flask API → ML model → results → UI
